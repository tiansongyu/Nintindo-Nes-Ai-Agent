# 视频素材目录

此目录用于存放视频制作所需的素材文件。

## 建议添加的素材

### 1. FCEUX截图
建议截取以下FCEUX工具界面截图：
- `fceux_main.png` - FCEUX主界面
- `ram_search_window.png` - RAM Search窗口
- `hex_editor.png` - Hex Editor窗口
- `cheat_window.png` - 作弊码窗口

### 2. 游戏截图
- 各个游戏的关键画面截图
- 显示生命值、分数等UI元素的截图

### 3. 图标和标识
- `nes_logo.png` - NES Logo
- `6502_chip.png` - 6502 CPU芯片图片

## 素材获取方法

### FCEUX截图
1. 打开FCEUX模拟器
2. 加载任意NES游戏
3. 打开对应工具窗口 (Tools菜单)
4. 使用截图工具截取窗口

### 游戏截图
1. 在FCEUX中运行游戏
2. 暂停在需要的画面
3. File → Screenshot 或按快捷键保存

## 素材规格建议
- 分辨率: 1920x1080 或更高
- 格式: PNG (无损) 或 JPG
- 命名: 使用小写英文和下划线
